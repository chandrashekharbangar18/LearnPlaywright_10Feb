# LearnPlaywright_10Feb

This repository contains code samples and exercises for learning Playwright, JavaScript basics, and related concepts.

## Structure

- `chapter_01_basics/` - Playwright and JavaScript basics
- `chapter_02_JSConcepts/` - JavaScript concepts and examples

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/chandrashekharbangar18/LearnPlaywright_10Feb.git
   ```
2. Install dependencies (if any):
   ```
   npm install
   ```
3. Explore the code in each chapter folder.

## Requirements
- Node.js (recommended)
- Playwright (if running Playwright scripts)
